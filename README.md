📌 Audiobook Generator (PDF to Speech)
This project is a desktop-based audiobook generator that converts PDF files into spoken audio using Python. It provides a simple and user-friendly graphical interface built with Tkinter, allowing users to select PDF files and listen to their content as audio.

The system works completely offline using text-to-speech technology, making it lightweight and accessible without requiring internet connectivity.

🎯 Features
📄 Convert PDF files into audio

🔊 Offline text-to-speech using pyttsx3

🖥️ Simple GUI built with Tkinter

📂 File selection dialog for easy usage

⚡ Lightweight and fast processing

🌐 Option to open links using webbrowser (if used)

🛠️ Tech Stack
Python

pyttsx3 – Text-to-speech engine

PyPDF2 – PDF text extraction

Tkinter – GUI interface

PIL (Pillow) – Image handling for UI

webbrowser – Opening external links

🧠 How It Works
User selects a PDF file using the file dialog

PyPDF2 extracts text from the document

Extracted text is passed to pyttsx3
